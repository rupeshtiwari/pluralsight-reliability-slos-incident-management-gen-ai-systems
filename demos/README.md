# Demo Index

This folder is a course-wide index of demos.

Do not run demos from this folder.  
Run each demo from its module directory. Nothing needs to be moved into `demos/`.

## Module 1: Production Reliability Fundamentals for GenAI Systems

### Clip 4: OpenTelemetry Health Checks and Synthetic Probes

**Demo path**

```bash
cd module-01-production-reliability-fundamentals/demo-clip4-otel-health-probes
```

**Start the demo**

```bash
./scripts/demo_up.sh
./scripts/open_ui.sh
./scripts/run_story.sh
```

**Stop the demo**

```bash
./scripts/demo_down.sh
```
